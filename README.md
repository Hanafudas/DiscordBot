# Install

1. Clone this repo

For SSH

`git clone git@github.hanafudas/discordbot.git`

For HTTPS

`git clone https://github.com/hanafudas/discordbot.git`

2. Run `npm i`

3. Replace token in `config.json`

4. Run `node index.js`
